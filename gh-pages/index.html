How the editor makes up its mind

A note on what is actually running on this page: no servers, no API calls, just mathematics in your browser.

The project compares three approaches to sarcasm detection, trained on the News Headlines Dataset for Sarcasm Detection (Misra & Arora): headlines from The Onion labelled sarcastic, and from HuffPost labelled sincere.

This page runs the logistic-regression baseline exactly. It uses the same 8,192 TF-IDF features and the same learned coefficients as the Python model in the repository, exported to JSON and evaluated in plain JavaScript. Its verdicts match the original model to machine precision, and it scores 84.6% accuracy (F1 0.84) on the 4,600 held-out test headlines.

Because logistic regression is linear, its reasoning is inspectable: every word carries a signed weight, and the marked-up copy above shows each word's actual contribution to the verdict. Words like “area”, “man” or “report” lean Onion; policy and lifestyle vocabulary leans HuffPost.

The two neural editors run here as well, once you summon them. The BiLSTM uses the exact weights from the repository, exported to ONNX, and scores 76.7% on the test set the way the Flask app serves it. DistilBERT was retrained in July 2026 with the repository's own training script, since the fine-tuned weights were never committed, and ships as an 8-bit quantized ONNX model to keep the download reasonable: 92.0% test accuracy, within half a point of the full-precision model. Its WordPiece tokenizer and VADER sentiment feature are ported exactly.
